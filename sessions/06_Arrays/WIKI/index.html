Creating Arrays

Arrays are generally described as "list-like objects"; they are basically single objects that contain multiple values stored in a list. Array objects can be stored in variables and dealt with in much the same way as any other type of value, the difference being that we can access each value inside the list individually, 

//Creating an Array
const data = [4,5,6];

Accessing Elements

Array Literals

An array literal is a list of zero or more expressions, each of which represents an array element, enclosed in square brackets ([]). When you create an array using an array literal, it is initialized with the specified values as its elements, and its length is set to the number of arguments specified. 

//Accessing Elements
let fish = ['Lion', , 'Angel'];

fish[0] is 'lion'
fish[1] is 'undefined'
fish[2] is 'Angel'

Measuring an Array

array.lendth

	fish.lenth;

Adding Elements to an Array

array.push() | array.unshift()

// Adding an element to the end
fish.push('shark')

//Adding an element to the beginning
fish.unshift('goldfish');
