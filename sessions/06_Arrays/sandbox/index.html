Arrays

An Array is a global object that is used to create an ordered list of collected data. Think of arrays as a list that can be any size you want it to be. Each item within that list can also hold a list of items as well.

Important things to remember about Arrays

  • The length of an array is not limited
  • The types of data an array can hold is not limited

        //Array Example One 
        const daysOfTheWeek = [
          "Monday",
          "Tuesday",
          "Wednesday",
          "Thursday",
          "Friday",
          "Saturday",
          "Sunday"
        ];

        //Array Example Two
        const fruits = ['Apple', 'Orange'];

        //Get array length
        console.log(fruit.length);

        //Array Example Three
        const scores = [10,30,15,25,40];
        console.log(scores[0])

        //

Array Methods

Arrays have dozens of methods that live/work on them. Methods are a way to describe functions that live on a JS object. Here are a few array methods below:

Push()

The push() method adds new items to the end of an array, and returns the updated array.


        const scores = [10,30,15,25,40];
        scores.push(60);

        console.log(scores);
        //logs [10,30,15,25,40,60]

Pop()

The pop() method removes the last element in an array


        const scores = [10,30,15,25,40];
        scores.push(60);
        scores.pop();

        console.log(scores);
        //logs [10,30,15,25,40]

Filter()

The filter() method creates a new array with all elements that pass the test ran by the provided function.


        const scores = [10,30,15,25,40];
        const filterScores = scores.filter((score)=> {
          return score > 20;
        });
